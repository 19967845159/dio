## 部署到 GitHub Pages

1. 将本仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 点击 Save，等待几分钟即可通过 `https://你的用户名.github.io/仓库名/` 访问
